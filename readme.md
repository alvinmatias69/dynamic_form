# Dynamic Form
A simple concept javascript script's to generate a dynamic form using a json data

## Usage
This script need a php to run it's server

```sh
$ git clone https://github.com/alvinmatias69/dynamic_form.git
$ cd dynamic_form
```

run the script :
```php
$ php -S localhost:8000
```
